Module Installation
NPM Package: https://www.npmjs.com/package/tealist

npm i tealist

Declaring Package
First, we need to declare the package. You can assign it any variable name.

const TeaListApi = require('tealist');
let api = new TeaListApi('API Key');

Quick Start

Example using Discord.js & an interval
Warning: Updating your users more than once per 10 minutes or faking information, will lead to key revocation & an API ban.

client.on('ready', () => {
  api.postServers(client.user.id, client.guilds.size);
  setInterval(function() {
    api.postServers(client.user.id, client.guilds.size);
  }, 900000); // This example posts your user & server counts once every 15 minutes.
});
